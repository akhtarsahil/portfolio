# Sahil Akhtar — Financial Analyst Portfolio

A modern, responsive, single-page static executive portfolio built with **HTML5**, **Tailwind CSS v3 (Play CDN)**, and interactive **JavaScript**.

## 🚀 Features

- **Hero Section**:
  - Headline: `'Financial Analyst | FP&A, Corporate Valuation & Behavioral Finance'`
  - Executive summary strip with live financial metrics ticker.
  - Customized high-resolution executive portrait (`assets/sahil_akhtar_portrait.png`) with floating KPI badge overlays.
- **About Section**:
  - Detailed breakdown of **3+ years of professional accounting experience** across **Accounts Payable (AP)**, **Accounts Receivable (AR)**, and **General Accounting**.
  - Highlighted **Dual Master's Degrees**:
    1. **Master's Degree in Business Finance** (Corporate Valuation, DCF, LBO, Capital Budgeting)
    2. **Master's Degree in Counseling Psychology** (Behavioral Finance, Cognitive Bias Mitigation, C-Suite Negotiation)
- **Projects Section (Interactive Case Studies)**:
  - **Live Interactive DCF Simulation Engine**: Sliders for Revenue Growth Rate (%), WACC (%), and Terminal Growth Rate (%) with real-time Enterprise Value & Implied Share Price calculations.
  - FP&A Budget vs. Actual Variance Engine case study.
  - Working Capital & DSO Reduction case study.
  - Behavioral Finance Cognitive Bias Audit toolkit.
- **Contact Section**:
  - Executive Contact Form with instant JavaScript feedback.
  - Direct email, location, and network links.
- **Dual Theme Support**:
  - Integrated **Light Mode / Dark Mode switcher** in the navigation bar.

## 🌐 How to Preview & Run

Simply double-click `index.html` to open it directly in any web browser (Chrome, Edge, Firefox, Safari). No installation or local server required!

Or serve locally via Python / Node:
```powershell
# Using Python
python -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.
